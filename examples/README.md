# arda examples

A tiny, self-contained demo. [`example.fasta`](example.fasta) holds five real
human GenBank mRNA records — one per locus (IGH, IGK, IGL, TRA, TRB) — and
[`example.airr.tsv`](example.airr.tsv) is arda's annotation of them (regenerated by
the command below).

## CLI

```bash
arda annotate -i examples/example.fasta -o /tmp/out.airr.tsv --organism human
```

Key columns of the output (one row per input sequence):

| sequence_id | locus | v_call | d_call | j_call | cdr3_aa |
|---|---|---|---|---|---|
| PZ235980.1 | IGH | IGHV3-9*01 | IGHD2-21*02 | IGHJ6*03 | ARDIGAGGFGDNFYFFYYMDV |
| PV083657.1 | IGK | IGKV1-33*01 | – | IGKJ5*01 | QQYDSLPYT |
| PP196642.1 | IGL | IGLV1-40*01 | – | IGLJ1*01 | QSYDSSLSGYV |
| PQ177856.1 | TRA | TRAV12-2*01 | – | TRAJ12*01 | AVNGDSSYKLI |
| AF043995.1 | TRB | TRBV28*02 | TRBD1*01 | TRBJ1-4*01 | ALSRGQAREKLF |

The full TSV has 49 AIRR columns: `v/d/d2/j_call`, `fwr1..fwr4`, `cdr1..cdr3` (+
`_start`/`_end` 1-based closed, `_aa`), `junction(_aa)`, `np1/np2/np3`,
`v_sequence_start/end`, `j_sequence_start`, `productive`, `rev_comp`, …

## Library

```python
import arda

records = arda.annotate_sequences(
    [("clone1", "GACGTGCAG..."), "CAGGTGCAGCTG..."],  # (id, seq) or bare strings
    seqtype="nt", organism="human",
)
for rec in records:
    print(rec["sequence_id"], rec["v_call"], rec["j_call"], rec["cdr3_aa"])
```

## Amino-acid input

```bash
arda annotate -i proteins.fasta -o /tmp/out.aa.airr.tsv --seqtype aa
```

## Cluster (SLURM) scale

```bash
# split → array-annotate → merge, chained via SLURM dependencies
arda slurm -i big.fastq -o big.airr.tsv --shards 50 --threads 8 --partition cpu
bash arda_slurm/submit.sh        # or: arda slurm ... --submit
```
